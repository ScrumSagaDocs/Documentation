Quick Start

GUI: Account Setup and Management

Users must have a ScrumSaga account before using the API. The account signup occurs, here At this time, users must also have a GitHub account. The API can perform ETL on any repo associated with the specific GitHub account. Take care that repos in GitHub Organizations will not be reachable unless the Organization is used as an account (??? correct).

Once a ScrumSaga account is created and the user is logged-in, all account managment is performed in the user menu portal. This provides a place for listing projects, diagrams, and previous work. It also displays users' current membership type and data limits.

API

This tutorial uses python with the ScrumSagaAPIWrapper. Additional language wrappers are currently in development. While certainly not necessary, Jupyter Notebooks are a very useful data science tool for multiple languages, and will be used by this site for creating reports.
